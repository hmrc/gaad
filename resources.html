Accessibility resources

Wear something purple on Thursday 16 May 2019 to mark the eighth Global Accessibility Awareness Day (GAAD).

Things to try

  • go mouse-free – try unplugging your mouse and only using your keyboard for an hour
  • set your browser zoom to 400% and check if the fonts resize – to find out if a page meets the Web Content Accessibility Guidelines
  • try some of the Windows accessibility features, like Windows Narrator
  • if you have an Apple device, try the Mac accessibility features
